# S&P 100 reviewed constituent source, 2026-07-18

This is a dated, maintainer-reviewed input for the explicit `sp100` import. Runtime code never downloads or scrapes this file.

## Provenance

- Index definition: [S&P Dow Jones Indices S&P 100](https://www.spglobal.com/spdji/en/indices/equity/sp-100/).
- Constituent extraction: the BlackRock iShares OEF fund-data download, retrieved 2026-07-18. OEF reports that it tracks the S&P 100 Index and its holdings file is dated 2026-07-16.
- Source file: `sp100-2026-07-18.csv`; source effective date: 2026-07-16; reviewer verification date: 2026-07-18.

## Review notes

- The source reports 101 equity securities. This is expected: Alphabet Class A (`GOOGL`) and Class C (`GOOG`) are distinct securities within the 100-company index.
- The BlackRock download labels Berkshire Hathaway Class B as `BRKB`. It is normalized here to the application ticker `BRK.B`, with the provider symbols `BRK-B`, matching Polygon stock aggregates.
- The file contains only active US equities on NYSE or NASDAQ; no ETF, index, cash, future, or derivative rows were copied.
